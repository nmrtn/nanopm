# mycelium — README snapshot
# Captured: 2026-06-26

## Tagline (unchanged)
"AI has made building cheap. It hasn't made deciding cheap."

## Core concept
AI agent framework for solo developers and small teams focused on deliberate decision-making before coding.

## Entry point
/mycelium:start — asks four questions (problems, stakeholders, risky assumptions, minimum viable tests) in ~10 minutes

## Two operating principles
- Scales: hierarchical decision levels from Purpose through Market
- Diamonds: repeating Discover-Define-Develop-Deliver loops

## Explicit anti-use cases
- Triage work
- Pure execution acceleration
- Centralized multi-role workflows
- Projects where discovery feels unnecessary
