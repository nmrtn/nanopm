# mycelium — README snapshot
# Captured: 2026-06-03

## Tagline (new)
> "AI has made building cheap. It hasn't made deciding cheap."

This is the cleanest articulation of the discovery-before-delivery thesis in the
landscape. Direct quote-worthy.

## What it is (new framing)
"AI agent framework that enforces discovery before delivery. Rather than jumping
from idea to code, it makes agents 'earn the right to start' by working through
structured decision-making phases before building."

## Positioning
- Target: builders (solo devs, small teams) where deciding what to build is harder
  than building it.

## Key features (refreshed)
- **Plugin-based installation for Claude Code** — brownfield-safe, no project-root
  modifications. Same convergent install pattern as prawduct + gstack.
- Guided 10-minute discovery via four core questions: problem, users, risks,
  smallest next move.
- Scales framework: Purpose → Strategy → Opportunity → Solution → Delivery → Market
- "Diamond" cycles (Discover-Define-Develop-Deliver) at each scale level
- Theory-gated progression requiring evidence, not confidence
- Canvas-based YAML specs committed to git as source of truth
- Adaptive intensity: hobby vs enterprise mode

## What makes it different (new section)
- Three interaction modes: **mentor** (guidance during phases), **guardrails**
  (at boundaries), **checklists** (at transitions). Shows contextually.
- EXPLICITLY rejects scope (triage work, pure execution, low-risk projects)
  rather than claiming universality.
- Regressions to earlier phases on bad assumptions = system SUCCESS, not failure.
- Implicit shot fired at "49 skills dumped simultaneously" framing (likely
  pointed at deanpeters/Product-Manager-Skills and adjacent skill catalogs).

## Strategic context vs prior snapshot (v0.23.30, 2026-05-21)
- Mycelium has crystallized its differentiation. Previously a complex 6-scale
  framework with theory gates; now positioned with a sharp tagline + explicit
  anti-positioning ("rather than 49 skills dumped").
- The "AI has made building cheap. It hasn't made deciding cheap." line is
  the most direct competitive headline in this space.
