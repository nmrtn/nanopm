# PM Skills: Overview

## What It Is

PM Skills is a comprehensive library of 68 AI-powered product management skills organized around the Triple Diamond framework. As stated on the homepage, these tools help "AI assistants produce professional PM artifacts - PRDs, user stories, acceptance criteria, experiment designs, sprint outputs, and more."

## Core Structure

The framework is built on **six sequential phases**:

1. **Discover** (5 skills) — Research and competitive analysis
2. **Define** (5 skills) — Problem framing and hypotheses
3. **Develop** (4 skills) — Solution briefs and architecture decisions
4. **Deliver** (6 skills) — PRDs, user stories, and launch checklists
5. **Measure** (6 skills) — Experiment design and results analysis
6. **Iterate** (4 skills) — Retrospectives and pivot decisions

Three additional skill families cut across these phases:
- **Foundation** (11 skills) — Personas, OKR writing, meeting lifecycle
- **Tool** (15 skills) — Facilitated sprint methodologies (Foundation and Design Sprints)
- **Utility** (12 skills) — Workflow building, validation, and library maintenance

## Key Features

**Getting Started**: Skills install via Claude Code's plugin marketplace or through open standards, then invoke by name (e.g., `deliver-prd`, `define-hypothesis`).

**Workflows**: 12 guided multi-skill sequences cover common PM processes like Feature Kickoff, Lean Startup, and Stakeholder Alignment.

**Quality Loop**: Three lifecycle utilities ensure consistency—Create, Validate, and Iterate skills manage the library as it grows.

**Portability**: Works on Claude Code, GitHub Copilot, Cursor, and any MCP-compatible client.
