## Pitch
Replace the 21-skill menu as first contact with a single zero-arg entry (`/pm-start`) that silently reads the founder's repo and runs ONE code-grounded challenge in under 90 seconds — returning the sentence ChatGPT structurally can't: "you're building X, but the job your code reveals is Y, and here's the question you're avoiding." Reframes the ask from "adopt a pipeline" to "let nanopm find the thing you're avoiding in your own product, right now." The pipeline, wiki, and viewer are never mentioned until after the aha lands.

## Riskiest assumption
That a single stateless first run — no wiki to compound against — can land as "I couldn't have gotten this from ChatGPT." If the differentiated value is actually the compounding state, a code-grounded punch may land generic or wrong, and a bad first impression is worse than silence.

## Cheapest test
Run the current `/pm-challenge-me` cold on 3–10 unfamiliar solo-founder repos; have outside founders rate each opening verdict "could ChatGPT have said this without my code? yes/no." Ship only if ≥7/10 are "no."

## Dissent/tension note
Design: a 21-skill menu is a museum map handed to someone who wanted a punch in the face — speed-to-aha is the whole product on run one. Business tension: if the hero skill becomes the product, you prove "one skill is useful" but never test whether the pipeline compounds — the actual bet.
