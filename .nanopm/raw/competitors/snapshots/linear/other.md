## Agent capabilities
- **Task delegation**: assign agents to issues, add them to projects, or @mention them in comments
- **Parallel work**: agents work across multiple issues simultaneously
- **Transparency**: inspect every change or reasoning trace
- **Accountability preservation**: human stays primary assignee, agent added as contributor

## Core positioning
"Agents as full members of your Linear workspace" — collaborative teammates, not replacements. "Natural collaboration" alongside delegation. Work can be delegated "end-to-end" while humans maintain oversight.

## Partner specializations
Coding, QA, PR management, root cause analysis, requirements writing, bug fixes.
