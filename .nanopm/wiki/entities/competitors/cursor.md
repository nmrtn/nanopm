## Summary
Cursor owns the coding-agent surface (Agents, Composer, Tab, Cloud Agents) and is commoditizing the skill-pack distribution model via its Teams marketplace. Its Composer planning loop and Cloud Agents reproduce the `/pm-prd → /pm-breakdown` sequence from inside the coding agent, but it ships no Define phase and no falsifiable-bet gate.

## What we know

Marketplace and Design Mode are now first-class; Cursor continues to commoditize skill-pack distribution. nanopm's moat must be the Define phase before code agents start, plus the adversarial gate.
