# Linear Agents Page — snapshot 2026-06-10

## What They Are
AI teammates working alongside humans in Linear's workspace as "artificial colleagues" for end-to-end task delegation.

## Core Capabilities
- Assignment to issues like regular team members
- @mentions in comments; contribution to projects
- Parallel work across multiple issues
- Transparent action logging
- Delegation while maintaining human accountability (humans stay primary assignees; agents are contributors)

## Listed Agent Integrations
- OpenAI Codex — parallel issue work, opens PRs
- Cursor — drafts branches ready for editor finishing
- GitHub Copilot — picks up assigned issues, creates PRs
- Devin — scopes issues, drafts PRs
- Sentry — root cause analysis via Seer
- ChatPRD — requirements writing, issue management
- Oz by Warp — answers questions, fixes bugs
- Factory — coding, testing, PR creation
- Charlie — TypeScript planning and review
- Ranger — bug formatting, test plan generation
- Tembo — delegates to any coding agent

## Use Cases
Code generation, technical task automation, QA, requirements documentation, bug fixing, root cause analysis.
