# PM Skills Marketplace: The AI Operating System for Better Product Decisions

> 68 PM skills and 42 chained workflows across 9 plugins. Claude Code, Cowork, and more. From discovery to strategy, execution, launch, growth, and shipping AI-built code.

Designed for Claude Code and Cowork. Skills compatible with other AI assistants.

## Why PM Skills Marketplace?

Generic AI gives you text. PM Skills Marketplace gives you structure. Each skill encodes proven PM frameworks—discovery, assumption mapping, prioritization, strategy—and walks through them step by step.

## How It Works (Skills, Commands, Plugins)

**Skills** are building blocks providing domain knowledge, analytical frameworks, or guided workflows for specific PM tasks. They load automatically when relevant.

**Commands** are user-triggered workflows invoked with `/command-name`, chaining multiple skills into end-to-end processes.

**Plugins** group related skills and commands into installable packages covering PM domains.

## Installation

### Claude Cowork

1. Open **Customize** (bottom-left)
2. Go to **Browse plugins** → **Personal** → **+**
3. Select **Add marketplace from GitHub**
4. Enter: `phuryn/pm-skills`

### Claude Code (CLI)

```bash
claude plugin marketplace add phuryn/pm-skills
claude plugin install pm-toolkit@pm-skills
```
