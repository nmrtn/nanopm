# AI Agent Skills for Product Managers

## Overview
PMPrompt offers 28 product management frameworks accessible as AI agent skills for Claude Code, Cursor, and OpenCode. The platform helps PMs automate workflows through reusable instruction files.

## Key Features

**Installation:** Users can run `npx skills add pmprompt/claude-plugin-product-management` to download all 28 skills locally.

## Available Skills

The platform includes frameworks for:
- **PRD Writer** - generates comprehensive product requirement documents
- **Working Backwards** - Amazon's press release approach
- **Shape Up** - Basecamp's methodology for defining scope and risks
- **Feature Prioritization** - uses RICE, Kano, and MoSCoW methods
- **Stakeholder Update Generator** - creates weekly team communications
- **Jobs-to-be-Done** - identifies functional, emotional, and social customer needs

## How It Works

1. Run installation command
2. Type skill commands in supported editors
3. Skills activate automatically for relevant tasks

## Evaluation Criteria

The site recommends selecting skills that offer process clarity, match specific work types (planning, discovery, prioritization, writing), and provide reusable commands.

## Additional Resources

The platform provides tools including a PRD Generator, User Story Generator, and Persona Generator, plus browsable categories covering product discovery, planning, execution, analytics, and marketing.

**Upgrade Option:** PMPrompt Pro offers unlimited documents, saved work organization, and 40+ included prompts.
